HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The !DOCTYPE declaration tells the browser what document type it should expect when it is opened.
  • The title element defines the title of a webpage.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external rescource to the HTML document.
  • The meta element defines metadata like keywords, viewports, and character sets.
  • The script element embeds client-side scripts in the HTML document.
  • The h1, h2, and similar elements represent the level of heading, creating larger or more prominent titles and subtitles to organize a page.
  • The p element represents a paragraph or block of text
  • The ul, ol, and li elements represent unordered lists, ordered lists, and list items respectively.
  • The list elements are used to make a list of reasons, similar to this list.
  • The img element contains information about images that are displayed on webpages.
  • The br element creats a line break between two blocks of content.
  • The a element creates links to webpages.

CSS

  • Inline CSS applies a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Internal CSS style sheets contain CSS rules for the webpage in the head section of the HTML file.
  • External CSS style sheets allows developers to keep all their CSS rules in a separate file, allowing it to affect multiple elements.
  • Selectors define the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property and the vaile of the property.
  • The margin property indicates how much space is around the outside of an element and it acts like a buffer.
  • The padding property adds space around the content inside an element.
  • The CSS box model views every HTML element as a box.
  • Each box consists of margins outside the element, the element's borders, paddigns inside the element, and the contents.
  • The * selector declared that all elements on the page will be affected by its style.
  • Using a comma allows to combine two selectors into a single CSS rule, allowing the declarations to affect both selectors.
  • Combining selectors helps with writing Do Not Repeat Yourself, or DRY, code.
  • The img selector assigns property values to any image element in the markup.

Git

  • Git is a version control system that keeps track of multipule versions of a project.
  • Git commands can be used in the Command Line Interface, or CLI, to save and track versions of code.
  • Using Git commands, you can initiate a repository, add code to the repository, and view different versions of the code, which are each stored in a unique commit.
  • There are approximately 37 Git commands to use.
